
# Hermes Agent

## Nykytila

Hermes Agent on hostattu Hostingerissa. Slack ja WhatsApp on yhdistetty. Hermes toimii henkilökohtaisena agenttina ja sitä ollaan rakentamassa Obsidian-pohjaiseksi toiseksi aivoksi.

## Mitä on tehty

- Hermes Agent käynnistetty Hostingerissa
- OpenRouter / Claude / Gemini -mallien käyttöä testattu
- Slack-integraatio yhdistetty
- WhatsApp-integraatio yhdistetty omaan numeroon
- Personal Obsidian Brain -skill luotu
- Obsidianiin luotu agentin muistirakenne

## Avoimet asiat

- Varmistaa, miten Hermes lukee ja kirjoittaa Obsidian-vaultiin
- Testata, että Slackista tai WhatsAppista lähetetty tieto menee oikeaan Obsidian-tiedostoon
- Luoda päivittäinen tai viikoittainen yhteenvetorutiini
- Määrittää mitä Hermes saa tehdä automaattisesti ja mitä vain luvalla

## Päätökset

- Hermes toimii henkilökohtaisena toisena aivona.
- Obsidian toimii pysyvänä muistina.
- Slack ja WhatsApp toimivat nopeina käyttöliittyminä.
- Epäselvät muistiinpanot menevät ensin `Inbox/Hermes inbox.md` -tiedostoon.
